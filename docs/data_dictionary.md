# Data Dictionary

- Crop_Yield_MT_per_HA: Crop yield per hectare (target variable)
- Average_Temperature_C: Average temperature
- Total_Precipitation_mm: Total rainfall
- Irrigation_Access_%: Percentage of irrigation access
- Fertilizer_Use_KG_per_HA: Fertilizer usage per hectare
- Economic_Impact_Million_USD: Economic impact (likely derived)
